Basic Forex Bot Usage Example (Python)

This example demonstrates how to initialize and start a basic Forex trading bot.


import time
from axiomtradeapi.client import AxiomTradeClient

# Replace with your actual API key and secret
API_KEY = "YOUR_API_KEY"
API_SECRET = "YOUR_API_SECRET"

# Initialize the client
client = AxiomTradeClient(api_key=API_KEY, api_secret=API_SECRET)

# Example: Using a BlackLotusX Basic Forex Bot
try:
    # This is a conceptual example. Actual bot initialization might vary.
    # Refer to specific bot documentation for exact usage.
    print("Initializing BlackLotusX Basic Forex Bot...")
    bot = client.forex_bots.get_basic_blacklotusx() # Assuming a method to get a specific bot instance

    # Configure the bot (example parameters)
    bot.configure(symbol="EURUSD", timeframe="H1", lot_size=0.01)

    print("Starting Forex Bot...")
    bot.start()
    print("Forex Bot started. Monitoring...")

    # Let the bot run for a while (in a real scenario, this would be a continuous process)
    time.sleep(60) # Run for 60 seconds as an example

    print("Stopping Forex Bot...")
    bot.stop()
    print("Forex Bot stopped.")

except Exception as e:
    print(f"An error occurred: {e}")

# Note: In a real application, bot management (starting, stopping, monitoring)
# would likely be handled in a more robust, long-running process.

How to run:

  1. Save the code as a Python file (e.g., forex_bot_example.py).
  2. Replace "YOUR_API_KEY" and "YOUR_API_SECRET" with your actual API credentials.
  3. Ensure you have the axiomtradeapi library installed (pip install axiomtradeapi - or the actual library name).
  4. Run the file from your terminal: python forex_bot_example.py

Fetch Account Balance Example (Python)

This example shows how to fetch your trading account balance using the API.


from axiomtradeapi.client import AxiomTradeClient

# Replace with your actual API key and secret
API_KEY = "YOUR_API_KEY"
API_SECRET = "YOUR_API_SECRET"

# Initialize the client
client = AxiomTradeClient(api_key=API_KEY, api_secret=API_SECRET)

try:
    print("Fetching account balance...")
    balance_data = client.get_account_balance() # Assuming a method to get balance
    print("Account Balance:")
    for currency, details in balance_data.items():
        print(f"- {currency}: Total = {details['total']}, Available = {details['available']}")

Retrieve Historical Data Example (Python)

This example shows how to fetch historical price data for a specific symbol and timeframe.

API_SECRET = "YOUR_API_SECRET"

# Initialize the client

try:
    print("Fetching historical data for BTCUSD H1...")
    # This is a conceptual example. Parameters might vary based on your API.
    historical_data = client.get_historical_data(
        symbol="BTCUSD",
        timeframe="H1",
        limit=100 # Fetch the last 100 candles
    )

    if historical_data:
        print(f"Fetched {len(historical_data)} historical data points.")
        # Example: Displaying the first 5 data points

            print(f"Candle {i+1}: Open={data['open']}, High={data['high']}, Low={data['low']}, Close={data['close']}, Volume={data['volume']}, Time={data['timestamp']}")

        # Optional: Convert to a pandas DataFrame for easier analysis
        # df = pd.DataFrame(historical_data)
