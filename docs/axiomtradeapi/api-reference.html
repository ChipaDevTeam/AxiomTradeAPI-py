AxiomTradeClient Class

The AxiomTradeClient class is the main client for interacting with the Axiom Trade API. It handles authentication and provides methods to access various functionalities of the platform.

__init__(self)

Purpose: Initializes the AxiomTradeClient.

Parameters: None.

Return Value: None.

login(self, email: str, b64_password: str, otp_code: str) -> Dict

Purpose: Completes the login process and stores authentication credentials.

Parameters:

  • email (str): The user's email address.
  • b64_password (str): The base64 encoded password.
  • otp_code (str): The one-time password for two-factor authentication.

Return Value: Dict - A dictionary containing the client credentials (likely including access and refresh tokens).

set_tokens(self, access_token: str = None, refresh_token: str = None)

Purpose: Sets access and refresh tokens manually.

Parameters:

  • access_token (str, optional): The access token to set. Defaults to None.
  • refresh_token (str, optional): The refresh token to set. Defaults to None.

Return Value: None.

refresh_access_token(self) -> str

Purpose: Refreshes the access token using the stored refresh token.

Parameters: None (uses the internally stored refresh token).

Return Value: str - The new access token.

Raises: ValueError if no refresh token is available.

get_trending_tokens(self, time_period: str = '1h') -> Dict

Purpose: Retrieves a list of trending meme tokens.

Parameters:

  • time_period (str, optional): The time period for trending data ('1h', '24h', '7d'). Defaults to '1h'.

Return Value: Dict - A dictionary containing trending token data.

Raises: ValueError if an access token is not set.

Example Usage:


client = AxiomTradeClient()
# Assume client is logged in and access_token is set
trending_data = client.get_trending_tokens('24h')
print(trending_data)

get_token_info(self, token_address: str) -> Dict

Purpose: Retrieves information about a specific token.

Parameters:

  • token_address (str): The address of the token.

Return Value: Dict - A dictionary containing token information.

Raises: ValueError if an access token is not set.

Example Usage:


client = AxiomTradeClient()
# Assume client is logged in and access_token is set
token_details = client.get_token_info('0x...') # Replace with a valid token address
print(token_details)

get_user_portfolio(self) -> Dict

Purpose: Retrieves the user's portfolio information.

Parameters: None.

Return Value: Dict - A dictionary containing the user's portfolio data.

Raises: ValueError if an access token is not set.

Example Usage:


client = AxiomTradeClient()
# Assume client is logged in and access_token is set
portfolio_data = client.get_user_portfolio()
print(portfolio_data)

Convenience Functions

quick_login_and_get_trending(email: str, b64_password: str, otp_code: str, time_period: str = '1h') -> Dict

Purpose: Provides a quick way to login and get trending tokens in a single call.

Parameters:

  • email (str): The user's email address.
  • b64_password (str): The base64 encoded password.
  • otp_code (str): The one-time password for two-factor authentication.
  • time_period (str, optional): The time period for trending data ('1h', '24h', '7d'). Defaults to '1h'.

Return Value: Dict - A dictionary containing trending token data.

Example Usage:


trending_data = quick_login_and_get_trending('your_email@example.com', 'your_b64_password', 'your_otp_code', '24h')
print(trending_data)

get_trending_with_token(access_token: str, time_period: str = '1h') -> Dict

Purpose: Provides a quick way to get trending tokens with an existing access token.

Parameters:

  • access_token (str): An existing valid access token.
  • time_period (str, optional): The time period for trending data ('1h', '24h', '7d'). Defaults to '1h'.

Return Value: Dict - A dictionary containing trending token data.

Example Usage:


# Assume you have a valid access token
trending_data = get_trending_with_token('your_access_token', '1h')
print(trending_data)
