Installation Guide for AxiomTradeAPI-py

This guide will walk you through the process of installing the AxiomTradeAPI-py Python library, allowing you to easily interact with the axiom.trade platform from your Python applications.

Prerequisites

Before installing the library, ensure you have the following installed on your system:

  • Python 3.6 or higher
  • pip (Python package installer), which usually comes bundled with Python.

Installation using pip

The recommended way to install AxiomTradeAPI-py is by using pip. If the library is available on a public package repository like PyPI, you can install it directly:

pip install axiomtradeapi

If you need to install a specific version, you can specify it:

pip install axiomtradeapi==X.Y.Z

(Replace X.Y.Z with the desired version number).

Installation from Source

If the library is not yet on a package repository, or if you want to install from the latest source code from a repository like GitHub, follow these steps:

  1. Clone the repository (replace <repository_url> with the actual URL): 
    git clone <repository_url>
  2. Navigate to the cloned directory: 
    cd <repository_directory>
  3. Install the library using pip in editable mode (for development) or standard install: 
    pip install .
    or 
    pip install -e .

Verifying Installation

To verify that the library has been installed correctly, open a Python interpreter and try importing the main client class:

from axiomtradeapi.client import AxiomTradeClient
print("AxiomTradeAPI-py installed successfully!")

If the import is successful and you see the confirmation message, the library is ready to use.
