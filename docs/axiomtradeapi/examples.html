Getting Started

Before running these examples, make sure you have installed the AxiomTradeAPI-py library. Refer to the Installation Guide for detailed instructions.

You will need your axiom.trade email, base64 encoded password, and potentially an OTP code for authentication.

Example 1: Logging In and Getting Trending Tokens

This example demonstrates how to log in to the axiom.trade platform and retrieve the current trending meme tokens.


import os
from axiomtradeapi.client import AxiomTradeClient

# Replace with your actual credentials
# For security, consider using environment variables or a config file
email = os.environ.get("AXIOM_EMAIL")
b64_password = os.environ.get("AXIOM_PASSWORD") # Your base64 encoded password
otp_code = os.environ.get("AXIOM_OTP", "") # Your OTP code (if 2FA is enabled)

if not email or not b64_password:
    print("Please set AXIOM_EMAIL and AXIOM_PASSWORD environment variables.")
else:
    try:
        client = AxiomTradeClient()
        client.login(email, b64_password, otp_code)
        print("Login successful!")

        trending_tokens = client.get_trending_tokens('24h')
        print("\nTrending Tokens (24h):")
        # Pretty print the JSON response
        import json
        print(json.dumps(trending_tokens, indent=4))

    except Exception as e:
        print(f"An error occurred: {e}")

How to run:

  1. Save the code as a Python file (e.g., get_trending.py).
  2. Set your axiom.trade email, base64 encoded password, and OTP (if applicable) as environment variables named AXIOM_EMAIL, AXIOM_PASSWORD, and AXIOM_OTP respectively.
  3. Run the file from your terminal: python get_trending.py

if not email or not b64_password:
    print("Please set AXIOM_EMAIL and AXIOM_PASSWORD environment variables.")
